## **Setup**
### Backend
1. Navigate to the `backend` folder:
   cd backend

2. Install dependencies:
    npm install

3. Run the backend:
    npm run dev

### Frontend
1. Navigate to the `frontend` folder:
    cd frontend

2. Install dependencies:
    npm install

3. Run the frontend:
    npm start
